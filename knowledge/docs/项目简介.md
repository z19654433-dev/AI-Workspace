# AI Agent 智能助手项目

## 项目概述
基于 DeepSeek API 的全栈 AI Agent 项目，实现了从 LLM 调用、意图识别到工具调用的完整 Agent 工作流。

## 技术栈
- Python 3.12 + FastAPI
- DeepSeek API（OpenAI 兼容 SDK）
- React 19 + Vite + TypeScript
- SQLite / ChromaDB

## 核心功能
1. Function Calling 自主工具调用：大模型自主决策调用天气查询、数学计算、热榜查询等工具
2. 装饰器工具注册：@registry.register() 一行注册，自动生成 JSON Schema
3. AST 安全求值器：替代 eval()，仅允许白名单内的数学运算
4. SQLite 长期记忆：多会话隔离，服务重启后上下文恢复
5. RAG 知识库：基于 ChromaDB 向量检索，支持文档内容问答
6. React 聊天界面：深色/白色主题，支持清空对话、复制消息

## 项目结构
分层架构：Agent 调度层 → Tools 工具层 → Memory 记忆层 → Chatbot API 层
前端独立为 frontend/ 目录，通过 API 与后端通信。
